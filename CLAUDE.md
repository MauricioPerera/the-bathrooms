# Claude Code — empiecen acá

Este repositorio usa la metodología **KDD (Knowledge-Driven Development)**. Las reglas obligatorias completas están en [.agents/AGENTS.md](.agents/AGENTS.md) y la skill local en [.agents/skills/kdd-okf-ccdd-hybrid/SKILL.md](.agents/skills/kdd-okf-ccdd-hybrid/SKILL.md). También existe un puente equivalente en [AGENTS.md](AGENTS.md).

Antes de escribir código: lee los contratos en `knowledge/contracts/` y valida con `python scripts/validate_contracts.py knowledge/contracts`. Este archivo es un puente, no duplica contenido.
